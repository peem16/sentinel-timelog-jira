          OAuth App (ตั้งครั้งเดียว)
          

            สร้าง app ครั้งเดียวเพื่อให้ปุ่ม Login ใช้งานได้ — callback URL ของทั้งคู่ใช้
            http://127.0.0.1:53817/callback

            • Atlassian: developer.atlassian.com → Create app (OAuth 2.0) → เปิด scope Jira read/write:jira-work, offline_access

            • Google: console.cloud.google.com → OAuth client (Desktop) + เปิด Calendar API (เลือก Internal ถ้าเป็น Workspace)

            • GitHub: github.com/settings/developers → New OAuth App → scope repo, read:org, read:user
          

          
          
          
          
          
          
        


        

          AI ช่วยเขียน Task/Bug
          

            เปิด/ปิดด้วยปุ่ม ✨ ในหน้าสร้าง Task — คำสั่งด้านล่างรับ prompt ทาง stdin แล้วพิมพ์คำตอบออก stdout
            (default ใช้ Claude Code ที่ login ไว้ในเครื่อง เปลี่ยนเป็น CLI อื่นได้ ถ้าหาไม่เจอให้ใส่ path เต็ม)
            โครงรายละเอียด/รูปแบบ issue ให้ skill ด้านล่างเป็นคนตัดสินใจ (ติดตั้ง skill ไว้ในเครื่องแล้ว)
            — default เปิด --allowedTools "Read Grep Glob Bash(git:*)" ให้ skill อ่านโค้ดจริง + ดึง git permalink (SHA) มาเติม Reference ได้ครบ (จำกัดเฉพาะ read/git ช้าลงหน่อยแต่ละเอียดกว่า)
          

          
          
          

            ทุกครั้งที่กด ✨ จะขึ้นต้น prompt ด้วย /<skill> เพื่อบังคับให้ Claude Code โหลด skill นี้แน่นอน (เว้นว่าง = ไม่ใช้ skill)
